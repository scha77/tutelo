## UAT: OG Image Platform Verification (S03)

- At least one teacher profile exists at a known slug (e.g. tutelo.app/ms-johnson or your test account slug)
- You have access to iMessage, WhatsApp, and a Facebook account for link paste tests
- Facebook Sharing Debugger: https://developers.facebook.com/tools/debug/

### Test 1: Unit Test Regression Guard

**Steps:**

### Test 2: Build Passes Clean

**Steps:**

**Expected:** Build completes with no errors

---

### Test 3: Facebook Sharing Debugger (og:url + og:image)

**Steps:**
1. Go to https://developers.facebook.com/tools/debug/
2. Enter a teacher profile URL (e.g. https://tutelo.app/[your-slug])
3. Click "Debug"
4. In the OG Tags table, confirm:
   - `og:url` = `https://tutelo.app/[slug]`
   - `og:title` = teacher name + "— Tutoring on Tutelo"
   - `og:description` = personalized description with subjects and location
   - `og:image` = an image URL (generated by opengraph-image.tsx)
   - `twitter:card` = `summary_large_image`
5. Click "Scrape Again" to force a fresh crawl
6. Confirm the preview card renders with teacher name, photo, and Tutelo branding

**Expected:** All OG tags present; preview card shows teacher name + image

---

### Test 4: iMessage Link Unfurl

**Steps:**
1. Open iMessage on your iPhone or Mac
2. Create a new message
3. Paste a teacher profile URL (https://tutelo.app/[slug])
4. Wait 2-3 seconds for the link preview to load

**Expected:** A link preview card appears showing the teacher's photo (or OG image), name, and "tutelo.app" domain — not a grey box or bare URL

---

### Test 5: WhatsApp Link Unfurl

**Steps:**
1. Open WhatsApp (mobile or desktop)
2. Start a new message in any chat
3. Paste the teacher profile URL
4. Wait for the link preview to load

**Expected:** Preview card shows teacher name, image, and description — consistent with the OG tags

### Test 6: Slack Unfurl (Optional)

### Edge Case: Slug Fallback

**Steps:**
1. Paste a non-existent teacher URL (e.g. https://tutelo.app/zzz-does-not-exist) into the Facebook Debugger
2. Confirm `og:url` is absent (the fallback metadata path returns `openGraph: undefined`)

**Expected:** No OG tags for invalid slugs — Facebook shows generic Tutelo title and description only
