Client-side grouping (run in the RSC page, not a client component):
```typescript
const studentMap = new Map<string, { name: string; subjects: Set<string>; count: number }>()
for (const b of completedBookings ?? []) {
  const key = `${b.student_name}|${b.parent_email}`
  const existing = studentMap.get(key)
  if (existing) {
    existing.subjects.add(b.subject)
    existing.count++
  } else {
    studentMap.set(key, { name: b.student_name, subjects: new Set([b.subject]), count: 1 })
  }
}
const students = Array.from(studentMap.values()).sort((a, b) => b.count - a.count)
```

Render a list of student rows: name (bold), subjects (comma-separated, muted), sessions count (e.g. "3 sessions"). Empty state: "No students yet. Sessions will appear here after they are completed."
